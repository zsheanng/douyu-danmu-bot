# douyu-danmu-bot
斗鱼直播间自动弹幕发送脚本，基于 Selenium 调用本地 Edge 浏览器，弹幕随机间隔 15~30 秒，降低平台风控检测概率。

> ⚠️ 风险声明：使用第三方自动化脚本发送弹幕违反斗鱼用户协议，本项目仅用于 Python 编程学习，请勿长时间挂机使用，账号封禁风险由使用者自行承担。

---

## 一、前置准备
### 1. 基础环境依赖
1. Python 3.10 及以上版本
2. Git（可选，用于代码版本管理、上传 GitHub）
3. 安装 selenium 依赖库
```powershell
pip install selenium
```
4. Microsoft Edge 浏览器（Windows 系统自带，无需额外下载）

### 2. 项目文件结构
项目目录 `danmuBot` 内必须包含两个核心文件：
- `main.py`：程序主运行脚本
- `danmu.json`：弹幕文案库（标准 JSON 数组格式）

### 3. danmu.json 标准模板
```json
[
    "666主播加油",
    "画面真不错",
    "学到了学到了",
    "太强了主播",
    "支持主播！"
]
```
> 格式规范：仅使用英文标点，文件编码选择 UTF-8，文件不可为空、不可写注释。

## 二、环境验证
### 1. 校验 Python 环境
```powershell
python --version
```
终端正常输出版本号即代表安装成功。

### 2. 校验 selenium 依赖
```powershell
pip show selenium
```
输出版本信息代表依赖安装完成。

### 3. 校验项目文件完整性
切换到项目目录，查看文件列表：
```powershell
cd E:\code\python\danmuBot
ls
```
列表中存在 `main.py`、`danmu.json` 即可正常运行。

## 三、运行使用教程
### 1. 启动脚本命令
在项目 PowerShell 终端执行：
```powershell
python -u "E:\code\python\danmuBot\main.py"
```

### 2. 登录操作完整流程
1. 脚本运行后自动弹出 Edge 浏览器，加载目标斗鱼直播间页面；
2. 页面弹出登录二维码，手机扫码完成登录；
3. 登录成功会自动跳转新标签页，等待页面全部加载完毕；
4. 回到 PowerShell 窗口按下回车，程序自动遍历所有标签，切回直播间页面并刷新同步登录状态。

### 3. 自动发送逻辑说明
- 循环读取 `danmu.json` 内所有文案逐条发送；
- 两条弹幕之间随机等待 **15 ~ 30 秒**，大幅降低风控检测；
- 终端实时打印待发送弹幕、剩余等待时长。

### 4. 终止程序
激活 PowerShell 终端窗口，按下快捷键：`Ctrl + C`
